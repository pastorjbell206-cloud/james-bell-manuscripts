## How This Library Is Used

- **Books and major manuscripts** ([01](01_Major_Books/), some entries in [09](09_Other/)) feed the Books section of [livewellbyjamesbell.co](https://www.livewellbyjamesbell.co/).
- **Booklets** ([03](03_Booklets/)) and **Hard Beliefs Series** ([04](04_HB_Series/)) become long-form articles or downloadable resources.
- **Articles** ([08](08_Articles/)) publish to the LiveWell Articles section.
- **PCN materials** ([05](05_PCN/)) feed the Pastors Connection Network resources.
- **FBC documents** ([06](06_FBC/)) and the **Resource Wall** ([07](07_ResourceWall/)) are pastoral / leadership resources.

## File Format

Each manuscript exists in two parallel files:

- `Name.docx` — the original Word document (authoritative source)
- `Name.md` — Markdown conversion with YAML front-matter (`source_docx`, `title_guess`, `word_count`, `converted_at`) for programmatic use

## Importing to the Website

A companion script — `scripts/import-manuscripts.mjs` (coming next) — maps each manuscript to the correct `posts` / `books` / `resources` table in the LiveWell database, generates slugs and excerpts, and inserts with idempotent upserts.
